Vision

To create a high-tech workforce that will help build Newark's Silicon valley.

Mission

To create the next generation of software engineers and to use coding as a tool to teach youth to be problem solvers, critical thinkers and collaborators.

Newark Kids Code is sponsored by The Urban League of Essex County. It is an educational initiative designed to introduce technologically disadvantaged and underrepresented youth to programming, (or "coding"), and other digital and computer and digital technologies.
